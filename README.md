# 课表管理系统

## 项目简介
这是一个基于PyQt6开发的现代化课表管理系统，专为高校学生设计。系统采用直观的网格布局展示课程信息，支持智能的时间冲突检测，提供丰富的数据导出选项和个性化的主题定制功能。通过简洁优雅的界面设计和人性化的交互体验，帮助学生高效管理日常课程安排。

## 功能特性

### 1. 课程管理
- **添加课程**
  - 双击空白格子快速添加
  - 支持批量导入课程
  - 智能时间冲突检测
  - 自动保存课程信息

- **编辑课程**
  - 右键菜单快速编辑
  - 支持拖拽调整时间
  - 实时更新显示
  - 修改历史记录

- **删除课程**
  - 支持单个删除
  - 支持批量删除
  - 删除确认提示
  - 支持撤销删除

### 2. 界面显示
- **课表视图**
  - 周视图（单周课程）
  - 总课表视图（所有课程）
  - 自适应布局
  - 响应式设计

- **课程卡片**
  - 显示课程名称
  - 显示教师信息
  - 显示教室位置
  - 显示周次信息
  - 悬停显示详情
  - 支持自定义颜色

### 3. 数据导出
- **PDF导出**
  - 支持A4纸张大小
  - 自动适应页面
  - 支持横向/纵向
  - 支持自定义页边距

- **PNG导出**
  - 高清图片导出
  - 支持自定义尺寸
  - 适合社交媒体分享
  - 支持透明背景

- **ICS导出**
  - 支持导出到系统日历
  - 支持设置提醒时间
  - 自动同步课程信息
  - 支持周期性课程

### 4. 主题定制
- **预设主题**
  - 默认主题（简约蓝）
  - 薰衣草主题（温柔紫）
  - 森林绿主题（自然绿）
  - 珊瑚粉主题（温暖粉）
  - 深海蓝主题（深邃蓝）

- **自定义选项**
  - 主要颜色
  - 背景颜色
  - 文字颜色
  - 边框样式
  - 卡片样式

### 5. 课程评价
- **评分系统**
  - 五星评分
  - 支持小数分值
  - 实时更新
  - 评分统计

- **反馈功能**
  - 文字反馈
  - 时间记录
  - 历史查看
  - 反馈修改

## 安装指南

### 1. 系统要求
- 操作系统：Windows 7+/macOS 10.14+/Linux
- Python版本：3.9或更高
- 存储空间：>100MB
- 内存要求：>2GB

### 2. 安装步骤
```bash
# 克隆项目
git clone https://github.com/lianshuang-photo/python-homework.git
cd course-manager

# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows
.\venv\Scripts\activate
# macOS/Linux
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 初始化数据库
python init_db.py

# 启动应用
python main.py
```

## 使用指南

### 1. 快速开始
1. 首次启动
   - 运行程序
   - 设置学期开始日期
   - 选择默认主题

2. 添加课程
   - 双击空白格子
   - 填写课程信息
   - 点击确认保存

### 2. 快捷键
- 通用操作
  - `Ctrl+N`: 新建课程
  - `Ctrl+F`: 搜索课程
  - `Ctrl+E`: 导出课表
  - `Ctrl+Z`: 撤销操作

- 视图切换
  - `Alt+←`: 上一周
  - `Alt+→`: 下一周
  - `Alt+H`: 返回当前周
  - `Alt+A`: 显示总课表

### 3. 高级功能
1. 数据备份
   - 自动备份
   - 手动备份
   - 备份恢复

2. 课表分享
   - 导出分享码
   - 扫码导入
   - 文件导入

## 常见问题解答

### 1. 安装问题
Q: 安装依赖失败怎么办？
A: 尝试以下步骤：
- 更新pip: `pip install --upgrade pip`
- 使用国内镜像源
- 检查Python版本兼容性

### 2. 使用问题
Q: 如何处理课程时间冲突？
A: 系统会自动检测并提示冲突，您可以：
- 调整课程时间
- 删除冲突课程
- 手动确认继续添加

## 技术栈
- **前端框架**：PyQt6
- **数据存储**：SQLite3
- **文件处理**
  - reportlab (PDF生成)
  - Pillow (图片处理)
  - icalendar (日历文件)
